# Приложение «Блогикум»

## Описание проекта

«Блогикум» — учебный Django-проект: блог с публикациями, категориями, комментариями и пользовательскими профилями.  
Проект реализован в рамках учебного спринта и использует стандартные возможности Django.

---

## Структура проекта

После клонирования репозитория структура проекта выглядит следующим образом:
```
Dev/
 └── django_sprint4/
     ├── .vscode/            Служебная папка редактора (опционально)
     ├── .git/               Служебные файлы Git
     ├── tests/              Тесты Практикума
     ├── venv/               Виртуальное окружение
     ├── blogicum/           Директория проекта
     │   ├── blog/           Приложение блога
     │   ├── pages/          Приложение статичных страниц
     │   ├── static/         Статические файлы (локально)
     │   ├── templates/      HTML-шаблоны проекта
     │   ├── blogicum/       Настройки проекта
     │   ├── db.sqlite3      База данных (может отсутствовать)
     │   └── manage.py
     ├── .gitignore
     ├── db.json             Фикстуры базы данных
     ├── LICENSE
     ├── pytest.ini
     ├── README.md
     ├── requirements.txt
     └── setup.cfg
```

## Установка и запуск проекта

### Клонирование репозитория

```
git clone <URL_репозитория>  
cd django_sprint4
```
---

### Создание виртуального окружения
```
Linux / macOS:  
python3 -m venv venv  
```
```
Windows:  
python -m venv venv
```
---

### Активация виртуального окружения
```
Linux / macOS:  
source venv/bin/activate  
```
```
Windows:  
source venv/Scripts/activate  
```
---

### Обновление pip
```
python -m pip install --upgrade pip
```
---

### Установка зависимостей
```
pip install -r requirements.txt
```
---

### Применение миграций
```
cd blogicum  
python manage.py migrate
```
---

### Запуск проекта
```
python manage.py runserver  
```
Проект будет доступен по адресу: http://127.0.0.1:8000/
